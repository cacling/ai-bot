# 推荐文案规范

## 风格要求

- 使用完整自然语言句子
- 口语化、亲切但不随意
- 以"帮我"或动词开头，表达用户视角

## 长度限制

- 中文：8-20 字
- 英文：5-15 词

## 禁用表达

- 避免标签词："查话费"、"查套餐"、"报修"
- 避免系统术语："skill"、"工单"、"路由"
- 避免过于笼统："查一下"、"帮我看看"（无上下文时）

## 正面示例

| 场景 | 推荐文案 |
|------|---------|
| 账单查询 | 帮我查一下本月账单明细 |
| 费用异常 | 帮我分析这月费用为什么变高 |
| 网络故障 | 最近网速很慢，帮我排查一下 |
| 套餐推荐 | 看看有没有更适合我的套餐 |
| 退订业务 | 帮我退订不需要的增值业务 |
| App 问题 | 营业厅 App 打不开，帮我看看 |
| 转人工 | 帮我转人工客服 |

## 双语规范

- 中英文案独立编写，不要直译
- 英文用祈使句或 "Help me..." 结构
