# 情绪识别详细指南

## 分类示例

### 平静 😌
- "我想查一下 2 月份的话费"
- "帮我看看套餐流量还剩多少"
- "好的，知道了"
- "那我知道了，谢谢"（功能性感谢，无明显情绪）

### 礼貌 🙏
- "谢谢你，辛苦了"
- "麻烦帮我查一下，谢谢"
- "没关系，你们已经很尽力了"
- "好的，感谢您的解答"

### 焦虑 😟
- "那我的流量会不会超？超了怎么办？"
- "我明天要用，来得及吗？能确定吗？"
- "这个会不会影响我的合同？我有点不确定"
- "是说今天就能恢复吗？还是要等很久？"

### 不满 😒
- "这个收费我觉得不合理"
- "之前说好的，现在又变了"
- "为什么我的流量用这么快，你们有没有多扣"
- "这个问题我已经反映过一次了"
- "说了半天还是没解决"

### 愤怒 😡
- "你们这服务太差了！"
- "我要投诉！这明显是乱收费！"
- "说了半天什么都没解决，你们有没有在认真处理！"
- "再不解决我就换运营商了！"
- "我受够了，叫你们领导来！"

## 边界判断

| 例句 | 正确标签 | 说明 |
|------|---------|------|
| "有点贵" | 不满 | 轻微不满，非愤怒 |
| "太贵了！怎么这么贵！" | 愤怒 | 感叹+强烈程度词 |
| "会不会有问题" | 焦虑 | 担忧式提问 |
| "谢谢，但这个价格确实贵了点" | 礼貌 | 主导情绪是礼貌，不满为次要 |
| "好吧，那就这样" | 平静 | 接受结果，无明显情绪 |
