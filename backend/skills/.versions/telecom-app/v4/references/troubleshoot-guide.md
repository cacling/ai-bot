# 营业厅 App 使用问题排查手册

> 本手册供客服参考，涵盖营业厅 App 五类常见问题的排查步骤与处理指引。
> 安全类诊断结论以 `diagnose_app` 工具输出为准，不得凭空推断。

---

## 一、问题类型速查

| 客户描述 | issue_type |
|---|---|
| "App 打不开"、"一打开就闪退"、"白屏/卡死" | `app_crash` |
| "登不进去"、"OTP 收不到"、"账号被锁" | `login_issue` |
| "查话费查不了"、"缴费页面报错"、"功能没反应" | `feature_error` |
| "装不上"、"更新失败"、"版本过低" | `install_update` |
| "设备不兼容"、"Root 检测"、"可疑登录提醒"、"账号被限制" | `security_check` |

---

## 二、TC1 · 闪退 / 无法打开（app_crash）

### 排查流程

```
#1 版本检查 → 过低 → 应用商店更新 → 结束
#2 清除缓存 → 重新打开
#3 重启设备 → 重新打开
#4 存储空间检查 → 不足 → 清理后重试
#5 全部无效 → 升级 frontline（设备型号 + 系统版本 + 截图）
```

### 操作步骤

**清除缓存（Android）**
设置 → 应用管理 → 营业厅 App → 存储 → 清除缓存 → 重新打开

**清除缓存（iOS）**
直接卸载 App，前往 App Store 重新下载安装

**检查存储空间**
- Android：设置 → 存储，查看可用空间（建议保留 1GB 以上）
- iOS：设置 → 通用 → iPhone 存储空间

## 三、TC2 · 登录问题（login_issue）

### 场景处理

| 场景 | 处理方式 |
|---|---|
| 密码错误（< 5次） | 提醒注意大小写/输入法，再试 |
| 密码错误（≥ 5次） | 停止尝试，使用"忘记密码"重置 |
| OTP 未收到 | 核验手机号 → 检查勿扰模式 → 等2分钟重发 → 切换验证方式 |
| 换了新手机 | 引导完成新设备注册 + OTP 验证 |
| 账号被临时锁定 | 24小时自动解锁，或调用 diagnose_app 人工核验后提前解锁 |
| 账号永久锁定 | 升级 security_team，引导携证件至营业厅申诉 |
| 风控/异地登录提醒 | 调用 diagnose_app(issue_type=suspicious_activity) |

### 密码重置步骤

打开 App → 登录页 → "忘记密码" → 输入手机号 → 接收 OTP → 设置新密码

### OTP 排查步骤

1. 确认手机号与注册号码一致
2. 检查手机是否处于勿扰模式
3. 等待 2 分钟后点击"重新发送"
4. 切换验证方式（短信 ↔ 语音验证码）
5. 仍无效 → 升级人工客服核身

## 四、TC3 · 功能异常（feature_error）

### 常见场景

| 功能 | 可能原因 | 处理方式 |
|---|---|---|
| 查话费/账单 | 网络异常、版本过低 | 切换网络，更新 App |
| 缴费 | 网络异常、账单系统维护 | 切换网络，等待后重试 |
| 办理业务 | 账户状态异常（欠费/停机） | 查询账户状态，引导先缴费 |
| 页面空白/报错 | 缓存问题 | 清除缓存后重进 |
| 所有功能不可用 | App 版本严重过低 | 强制更新至最新版 |

### 排查顺序

1. **切换网络**：Wi-Fi ↔ 移动数据，重试
2. **检查版本**：应用商店查看是否有更新
3. **清缓存重启**：清除缓存 → 强制停止 → 重新打开
4. **查账户状态**：是否欠费/停机导致功能受限
5. **以上无效**：升级 frontline，记录问题功能 + 错误截图

## 五、TC4 · 安装 / 更新问题（install_update）

### 常见场景

| 问题 | 原因 | 处理方式 |
|---|---|---|
| 应用商店找不到 | 地区限制 | 提供官方下载渠道 |
| 安装失败 | 存储不足 / 系统版本过低 | 清理空间 / 升级系统 |
| 更新失败 | 网络不稳定 | 切换网络重试 |
| 安装后无法打开 | 安装包损坏 | 卸载重新下载安装 |

### 系统版本要求

- Android：建议 Android 8.0 及以上
- iOS：建议 iOS 13.0 及以上

### 官方下载渠道

- 华为应用市场 / 小米应用商店 / OPPO 软件商店 / vivo 应用商店
- 苹果 App Store（搜索"营业厅"）
- 官网扫码下载（Android APK）

## 六、TC5 · 设备 / 账号安全（security_check）

### App 被锁定排查（TC5-A 四问）

| 检查项 | 诊断信号 | 处理 |
|---|---|---|
| App 版本 | 版本过低 | 更新至最新版 |
| 近期安装应用 | 命中黑名单 | 删除可疑应用 → 重新登录 |
| 不熟悉的应用 | 命中黑名单 | 删除可疑应用 → 重新登录 |
| 设备安全 | Root/模拟器/屏幕共享 | 对应处理（见下） |

### 设备安全状态处理

| 状态 | 处理 |
|---|---|
| Root / 越狱 | 硬性限制，须使用正常设备 |
| 开发者模式 | 引导关闭后重试 |
| 模拟器环境 | 硬性限制，须在实体手机上使用 |
| 虚假 GPS | 引导卸载相关应用 |
| VPN / 代理 | 引导关闭后重试 |
| 远控 / 屏幕共享 | 立即停止，升级 security_team |

### 反诈提醒话术

> 如客户设备上存在屏幕共享、远程控制应用，且客户声称是被他人指导操作的：
>
> "您好，请注意，当前情况符合常见电信诈骗的操作模式。请您立即：
> ①停止与对方的一切通话/聊天；
> ②关闭所有远程控制应用；
> ③不要向对方透露任何验证码或个人信息。
> 我们将协助您保护账号安全。"

## 七、升级路径说明

| 升级路径 | 适用场景 | 处理时效 |
|---|---|---|
| `self_service` | 客户可自助完成（更新 App、清缓存、切网络等） | 即时 |
| `frontline` | 需客服介入（截图审查、人工解锁、工单提交）| 当天处理 |
| `security_team` | 高风险（Root/永久锁定/诈骗嫌疑） | 1 个工作日内 |

---

## 八、紧急联系

- **客服热线**：App 内 → 联系客服（24 小时）
- **安全事件报告**：App 内 → 设置 → 安全中心 → 举报可疑活动
- **账号挂失**：拨打客服热线，告知"账号可能被盗，申请紧急挂失"
