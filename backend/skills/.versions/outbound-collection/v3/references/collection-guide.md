# 外呼催收话术手册

> 本手册供外呼机器人话术参考，所有账单数据以案件系统下发信息为准，不得凭空填写。

---

## 一、开场白（接通后立即播放）

要点：自我介绍（运营商客服小通） → 告知本通话可能被录音 → 确认客户身份

---

## 二、身份核验

### 核验项
- 姓名确认
- 证件后四位
- 出生日期
- 注册手机号后四位

### 核验失败
告知无法核实身份 → 建议客户本人拨打客服热线 10086 → 礼貌道别

### 非本人接听
致歉打扰 → 请帮忙转告机主有重要事项需要联系 → 请回拨 10086 → 道别

---

## 三、逾期告知

### 标准告知（轻催策略）
告知产品名称、欠款金额、逾期天数 → 提醒为避免影响正常使用请尽快缴费 → 询问是否方便今天处理

### 加重告知（中催策略）
告知产品名称、欠款金额、逾期天数 → 提醒持续逾期可能影响信用记录 → 询问近期是否有安排还款

### 强催话术（强催策略）
告知产品名称、欠款金额、逾期天数 → 说明情况较为紧急，将依照合同约定采取措施 → 询问今天能否安排还款

---

## 四、各意向应答

### 4.1 承诺还款（PTP）

**客户表示会还款时：**
感谢配合 → 确认具体还款日期 → 告知将发送缴费链接短信 → 感谢接听并道别

**确认还款方式：**
- 营业厅 App 缴费
- 微信/支付宝扫码
- 营业厅窗口

---

### 4.2 明确拒绝（refusal）

**初次拒绝应答：**
表示理解客户困难 → 提醒持续逾期可能影响信用评分和账号使用 → 建议尽快处理 → 告知可以提供协助

**客户再次拒绝，不转人工时：**
记录客户情况 → 告知后续会再次联系 → 感谢接听并道别

**触发转人工时：**
表示理解客户情绪 → 告知正在转接专属客服 → 请客户稍候

---

### 4.3 提出异议（dispute）

**已还款异议：**
感谢告知 → 询问还款渠道和时间 → 告知财务团队 1-3 个工作日内核实 → 核实结果短信通知 → 如有疑问请拨打 10086

**金额有误异议：**
表示理解 → 告知已记录异议并提交复核 → 复核时限 3 个工作日 → 结果短信告知 → 紧急情况请拨打 10086

**非本人借款：**
致歉困扰 → 引导拨打 10086 申诉 → 说明会启动异议处理流程保障权益

---

### 4.4 主动要求转人工

告知正在转接人工客服 → 请客户稍候

---

## 五、结束语

### 正常结束（还款承诺/核查安排后）
感谢配合 → 如有问题拨打 10086 → 祝生活愉快并道别

### 非配合结束（拒绝/无法核验）
感谢接听 → 道别

---

## 六、合规禁用语

| 禁止使用 | 原因 |
|---|---|
| 强迫性还款要求 | 强迫性语言，违规 |
| 不实的黑名单威胁 | 不实威胁 |
| 通知家人/单位 | 侵犯隐私 |
| 侮辱性语言 | 违反服务规范 |
| 连续拨打超过策略限制次数 | 骚扰行为 |

---

## 七、通话结果分类

| result 值 | 含义 |
|---|---|
| `ptp` | 承诺还款，已确认日期 |
| `refusal` | 明确拒绝还款 |
| `dispute` | 提出异议（已还/金额错/非本人） |
| `transfer` | 已转人工 |
| `verify_failed` | 身份核验失败 |
| `non_owner` | 非本人接听 |
| `no_answer` | 未接通（无应答） |
| `busy` | 忙线 |
| `power_off` | 关机/停机 |
| `callback_request` | 客户要求回电 |
