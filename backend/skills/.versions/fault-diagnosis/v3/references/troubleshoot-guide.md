# 网络故障排查手册

## 故障类型及处理步骤

### 1. 无信号（no_signal）

**常见原因**：
- SIM 卡未插好或松动
- 手机设置为飞行模式
- APN 配置错误
- SIM 卡损坏
- 所在区域无基站覆盖

**用户自查步骤**：
1. 检查是否处于飞行模式，关闭后重启网络
2. 取出并重新插入 SIM 卡
3. 重启手机
4. 检查 APN 设置（设置 → 移动网络 → 接入点名称），恢复默认
5. 换一个位置（如走到户外或窗边）

**仍无法解决**：
- 携带身份证至营业厅检测 SIM 卡
- 登录官网查询当前区域基站覆盖状态

---

### 2. 网速慢（slow_data）

**常见原因**：
- 套餐流量接近用尽（降速限流）
- 所在区域网络拥塞（晚高峰）
- 后台应用大量占用流量
- 手机信号强度不足

**用户自查步骤**：
1. 查看流量剩余（APP → 流量使用情况）
2. 关闭后台不必要的应用
3. 切换至 Wi-Fi 测试是否是移动网络问题
4. 尝试切换至 4G/5G 模式（设置 → 移动网络 → 首选网络类型）
5. 如流量耗尽，可购买临时流量包（APP → 加油包）

**网速参考标准**：
| 应用类型 | 最低所需速度 |
|---|---|
| 微信文字/语音 | > 0.5 Mbps |
| 标清视频 | > 1 Mbps |
| 高清视频 | > 3 Mbps |
| 4K 视频 | > 15 Mbps |

---

### 3. 通话中断（call_drop）

**常见原因**：
- 移动过程中信号不稳定（基站切换失败）
- 所在区域信号覆盖差
- VoLTE 设置问题

**用户自查步骤**：
1. 检查手机是否支持并开启了 VoLTE（4G 通话）
2. 尝试在信号较强的位置拨打
3. 重启手机后重试
4. 拨打 10086 测试通话质量（由客服侧检测基站）

**VoLTE 开启方法**：
- iOS：设置 → 蜂窝网络 → 启用 LTE → 语音和数据
- Android：设置 → 移动网络 → 启用 VoLTE 通话（各品牌位置略有差异）

---

### 4. 无法上网（no_network）

**常见原因**：
- APN 配置错误（有信号但连不上网）
- 账号欠费停机
- 后台应用限制（省电模式/流量限制）
- 流量达到当日封顶（部分套餐有日封顶）

**用户自查步骤**：
1. 检查账号余额（欠费会停止数据服务）
2. 关闭再打开移动数据开关
3. 重置 APN：设置 → 移动网络 → APN → 右上角菜单 → 重置为默认
4. 关闭省电模式
5. 重启手机

---

## 升级处理场景

以下情况无法通过自助排查解决，需提交人工工单：

| 场景 | 处理方式 |
|---|---|
| 区域内多用户集中反馈无信号 | 提交基站故障工单（预计 4 小时内响应） |
| SIM 卡疑似损坏 | 前往营业厅更换 SIM 卡（免费补卡一次） |
| 连续 3 次重启仍无信号 | 电话 10086 由技术支持远程检测 |
| 漫游场景无法使用 | 联系客服确认漫游协议是否覆盖当前区域 |

---

## 紧急联系方式

- **客服热线**：10086（24小时）
- **故障报修**：APP → 服务 → 网络故障报修
- **营业厅**：支持查询离你最近的营业厅地址
