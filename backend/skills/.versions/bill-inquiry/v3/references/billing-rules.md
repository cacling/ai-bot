# 电信计费规则参考文档

> 账单查询 Skill 的详细计费规则和处理指引知识库。

---

## 账单结构说明

每月账单由以下几项构成：

| 费用项 | 说明 |
|---|---|
| 套餐月费 | 固定收取，按月计费，月初扣费 |
| 流量超额费 | 套餐内流量用尽后产生，按 0.1元/MB 计费 |
| 通话超额费 | 套餐内分钟用尽后产生，按 0.1元/分钟 计费 |
| 增值业务费 | 已订阅的增值包月费，按月扣除 |
| 税费 | 按实际消费金额的 6% 征收增值税 |

## 计费周期

- **账期**：每月 1 日 00:00 至 当月最后一天 23:59
- **出账日**：次月 1 日
- **缴费截止**：出账日起 15 天内（超期将停机）
- **停机恢复**：缴清欠费后 30 分钟内自动恢复

## 流量计费规则

- 套餐流量优先使用，用完后进入按量计费
- 流量按 MB 向上取整计费
- 夜间免费流量（00:00-08:00）：部分套餐享有，具体以套餐说明为准
- 流量每月清零，不累计至下月

## 通话计费规则

- 套餐内分钟优先使用
- 超出部分：主叫 0.1元/分钟（按6秒为1个计费单元）
- 被叫：全部套餐免费
- 跨省漫游通话：仅"畅享 100G 套餐"及以上免费，其余套餐 0.2元/分钟

## 欠费停机说明

- 账户余额低于 0 且欠费超过 15 天，系统自动停机
- 停机后仍可接收来电（被叫免费）
- 停机期间流量和主叫功能均暂停
- 充值后余额大于 0 时，30 分钟内自动恢复

## 发票申请规则

- 仅缴清账单后方可申请发票
- 个人用户：增值税普通发票，抬头为手机号户名
- 企业用户：增值税专用发票，需提供企业名称、税号、地址、电话、开户行信息
- 申请渠道：官方 APP → 账单 → 申请发票
- 开具时效：提交后 3-5 个工作日，以电子邮件形式发送
- 补开历史发票：最多可补开近 12 个月账单发票

## 费用解读指引

向用户解释各费用项时，使用以下通俗说明模板：

| 费用项 | 解释模板 |
|---|---|
| 套餐月费 | "这是您每月固定的套餐费用，金额为 {X} 元，每月初自动扣除。" |
| 流量超额费 | "您的套餐流量已用完，超出部分按每 MB 0.1 元计费（不足 1MB 按 1MB 算），本月超额流量共 {X}MB，费用 {Y} 元。" |
| 通话超额费 | "您的套餐通话分钟已用完，超出部分按每分钟 0.1 元计费（以 6 秒为一个计费单元），本月超额通话 {X} 分钟，费用 {Y} 元。" |
| 增值业务费 | "您当前订购了以下增值业务：{业务名1}（{月费1}元/月）、{业务名2}（{月费2}元/月），合计 {Z} 元，需逐项向用户说明每项业务的名称和月费。" |
| 税费 | "根据国家规定，通信服务需缴纳 6% 的增值税，本月税费为 {X} 元，按实际消费金额计算。" |

**解读要点**：

- 先报总金额，再逐项拆解
- 每项费用给出具体数字，不使用"大约"、"差不多"等模糊表述
- 如用户对某项费用有疑问，结合对应计费规则章节详细解释

## 欠费处理指引

处理欠费用户的标准步骤：

### 第一步：确认停机状态

通过 `query_subscriber(phone)` 查询账号状态，确认是否已停机。

### 第二步：告知欠费详情

告知用户具体欠费金额和欠费月份，例如："您当前欠费 {X} 元，涉及 {月份1}、{月份2} 共 {N} 个月的账单。"

### 第三步：引导充值

告知用户可用的充值方式：

| 充值方式 | 操作说明 |
|---------|---------|
| APP 缴费 | 打开官方 APP → 首页"充值缴费" → 输入金额 → 支付 |
| 微信/支付宝扫码 | 打开微信或支付宝 → 搜索"中国XX电信" → 进入生活缴费 → 输入手机号缴费 |
| 营业厅窗口 | 携带身份证前往就近营业厅，现金或刷卡缴费 |

### 第四步：说明恢复时效

充值成功且余额大于 0 后，30 分钟内系统自动恢复通信服务，无需额外操作。

### 第五步：推荐自动缴费

建议用户开通自动缴费功能，避免忘缴导致停机。操作路径：APP → 我的 → 自动缴费 → 绑定银行卡或微信支付。

### 停机阈值规则

| 条件 | 系统动作 |
|------|---------|
| 余额低于 0 | 发送欠费短信提醒 |
| 欠费超过 15 天 | 自动停机（暂停主叫和流量，保留被叫） |
| 欠费超过 90 天 | 进入预销号状态，需到营业厅办理恢复 |
| 欠费超过 180 天 | 号码回收，无法恢复 |

## 发票申请指引

### 前提条件

用户必须已缴清所申请月份的账单，未缴清的账单无法开具发票。

### 申请渠道

官方 APP → 底部"账单"标签 → 选择月份 → 点击"申请发票"。

### 发票类型

| 用户类型 | 发票类型 | 所需信息 |
|---------|---------|---------|
| 个人用户 | 增值税普通发票 | 无需额外信息，抬头默认为手机号户名 |
| 企业用户 | 增值税专用发票 | 企业名称、统一社会信用代码（税号）、注册地址、联系电话、开户银行及账号 |

### 开具时效

提交申请后 3-5 个工作日内以电子邮件形式发送至用户注册邮箱。

### 补开规则

- 可补开近 12 个月内已缴清的账单发票
- 超过 12 个月的账单无法补开，需前往营业厅申请纸质证明
- 同一月份账单仅可开具一次发票，如需重新开具需先作废原发票

## 异常费用分析

当用户反馈话费异常偏高时，按以下步骤排查：

### 分析步骤

1. **拉取当月和上月账单**：通过 `query_bill(phone, month)` 分别获取当月和上月账单
2. **对比总金额**：计算差额，确认是否确实存在显著增长（超过 20% 视为异常）
3. **逐项对比**：找出金额变化最大的费用项
4. **检查流量超额**：对比套餐内流量和实际用量，确认是否因流量超额导致
5. **检查增值业务**：对比两月的增值业务列表，确认是否有新增订阅
6. **检查通话超额**：对比套餐内分钟数和实际通话时长
7. **无法定位原因**：如以上均无法解释差异，升级至 10086 人工客服处理

### 常见异常模式

| 异常模式 | 典型表现 | 原因分析 | 建议操作 |
|---------|---------|---------|---------|
| 流量超额 | 流量超额费从 0 元突增至数十元 | 套餐流量用尽后继续使用移动数据 | 建议购买流量加油包或升级套餐 |
| 新增增值业务 | 出现之前没有的增值业务扣费项 | 用户可能误订或被推广订阅 | 引导用户在 APP 查看已订业务并退订 |
| 通话超额 | 通话超额费显著增加 | 当月通话时长超出套餐包含分钟数 | 建议购买通话加油包或升级套餐 |
| 漫游费用 | 出现跨省漫游通话费 | 用户在外省使用了通话功能，套餐不含漫游 | 说明漫游计费规则，建议升级含漫游的套餐 |
| 原因不明 | 各项费用均无明显变化但总额偏高 | 可能涉及系统计费异常或隐性扣费 | 升级至 10086 热线，由人工核查计费记录 |

### 分析话术模板

- 存在超额："经对比，您本月账单比上月高出 {X} 元，主要原因是{费用项}超出套餐额度，超额部分产生了 {Y} 元费用。"
- 新增业务："您本月新增了一项增值业务"{业务名}"，月费 {X} 元。如非本人订阅，可在 APP 中退订。"
- 无法定位："目前无法确定费用异常的具体原因，建议您拨打 10086 由人工客服进一步核查。"
