## 二、开场话术要点

### 标准开场
问候客户并确认身份 → 自我介绍（电信客服小通） → 告知本通话可能被录音 → 说明来电目的（介绍专属套餐优惠） → 询问是否方便

### 客户说忙/不方便
致歉打扰 → 询问方便回联的具体时间 → 告知届时会回电

### 客户同意听
感谢客户 → 提及客户当前套餐名称 → 指出可能的痛点（流量/通话不够用或仍在4G） → 引出推荐方案

---

## 三、异议处理要点

### 异议1：价格太贵
理解客户顾虑 → 对比当前套餐费用加超额部分 → 说明升级后反而可能更省 → 询问是否可以接受

### 异议2：现在套餐够用
认可客户看法 → 询问是否遇到过网速慢的场景 → 说明5G和4G在速度上的差异（尤其高峰时段） → 引导体验

### 异议3：还在合约期
说明套餐升档可立即生效不影响现有合约 → 合约到期后可灵活降档

### 异议4：要去营业厅办
说明手机上即可直接办理 → 告知将发送短信包含套餐详情和办理链接

### 异议5：要和家人商量
肯定客户谨慎态度 → 先发套餐详情短信供参考 → 约定回访时间

---

## 四、促成要点

### 促成办理
确认客户办理方式偏好（APP / 后台登记） → 发送包含套餐详情和一键办理链接的短信 → 告知操作简单几分钟完成 → 如有问题拨打客服热线

### 回访预约
确认回访时间 → 告知届时回电 → 如有疑问欢迎主动拨打客服热线 → 感谢接听并道别

---

## 五、结束要点

### 成功转化
感谢信任 → 告知生效时间 → 届时会收到确认短信 → 如有疑问欢迎联系 → 祝使用愉快并道别

### 拒绝结束
表示理解 → 如有需要欢迎随时拨打客服热线 → 感谢接听并道别

## 六、合规禁用语

| 禁用语 | 替代表达 |
|---|---|
| 不实的涨价威胁 | 说明当前活动有效期，不可威胁 |
| 虚假的最后机会声明 | 说明活动目前还有名额，建议尽快考虑 |
| 保证省钱 | 根据用量估算可能节省的金额 |
| 引导跳过合同确认 | 不可引导用户跳过合同确认 |
| 免费升级（实际有费用） | 必须明确告知月租价格 |

## 七、通话结果分类

| result 值 | 含义 |
|---|---|
| `converted` | 客户同意办理，已引导完成 |

| `not_interested` | 客户明确拒绝，不感兴趣 |
