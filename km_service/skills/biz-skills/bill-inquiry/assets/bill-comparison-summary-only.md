# 账单对比仅总额模板

适用条件：

- 当前月账单存在
- 对比月账单存在
- 只能确认总额变化，不能稳定定位具体 item 级变化

推荐话术骨架：

```text
我先帮您对比了两个月的账单总额：

- {{previous_month}}：¥{{previous_total}}
- {{current_month}}：¥{{current_total}}
- 变化：¥{{diff}}

目前可以确认总额有变化，但我这次拿到的明细还不足以准确判断是哪个具体项目上涨，所以我先不直接给您下“哪一项涨了多少”的结论。

如果您愿意，我可以继续帮您核查更细的账单项目，或告诉您下一步该怎么处理。
```

约束：

- 禁止输出“从 ¥X 变为 ¥Y”的 item 级结论
- 禁止用“可能是某某业务”来替代真实证据，除非明确标注为“暂时无法确认”
