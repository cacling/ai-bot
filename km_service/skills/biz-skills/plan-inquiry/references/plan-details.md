## 套餐变更规则

### 升级套餐（月费增加）
- 申请后**立即生效**
- 当月费用：按天比例计算差价（已用天数按原套餐，剩余天数按新套餐）
- 示例：月中升级，当月补缴约半价差价

### 降级套餐（月费减少）
- 申请后**次月1日生效**
- 本月按原套餐收费
- 如已有超额费用，不因降级变化

### 注意事项
- 同一号码每月最多变更套餐 **2次**
- 合约期内的套餐可能有违约金，请提前确认
- 变更方式：APP → 我的 → 套餐变更；或到营业厅办理

## 套餐变更指引

> 详细的套餐变更规则和操作流程，供客服引导用户时参考。

### 升级规则（月费增加）

- **生效时间**：申请后立即生效
- **当月费用计算**：按天补差价
  - 计算公式：`补缴金额 = (新套餐月费 - 原套餐月费) × (当月剩余天数 / 当月总天数)`
  - 示例：原套餐 ¥50，升级到 ¥88，当月还剩 15 天（共 30 天），补缴 = (88 - 50) × (15 / 30) = ¥19
- **权益变化**：升级后立即享受新套餐的流量、通话等权益

### 降级规则（月费减少）

- **生效时间**：次月1日生效
- **当月费用**：本月仍按原套餐价格收费，享受原套餐权益
- **注意**：本月已产生的超额费用不因降级而减免

### 变更限制

- 每月最多变更 **2次**（含升级和降级）
- 合约期内变更可能产生违约金，需提前告知用户确认
- 合约剩余期限和违约金金额可通过 `query_subscriber(phone)` 查询

### 变更办理方式

| 方式 | 路径 | 适用场景 |
|------|------|---------|
| APP 自助 | APP → 我的 → 套餐变更 | 普通升级/降级，无合约限制 |
| 营业厅 | 携带身份证前往 | 合约期内变更、需解除违约金 |

### 用户确认要求

变更套餐前**必须**获得用户明确同意，需确认以下信息：
1. 新套餐名称和月费
2. 生效时间（升级立即/降级次月）
3. 当月补缴金额（如为升级）
4. 合约违约金（如有）

## 套餐对比指引

> 对比套餐时的标准展示格式和注意事项。

### 四维对比格式

对比任意两个或多个套餐时，统一使用以下四个维度：

| 维度 | 说明 |
|------|------|
| **月费** | 每月固定费用 |
| **流量** | 国内通用流量额度 |
| **通话** | 国内主叫分钟数 |
| **权益** | 附加权益（漫游、会员、热点等） |

### 对比展示要求

1. **突出差异项**：两个套餐相同的部分简略带过，不同的部分加粗或特别说明
2. **附带每GB单价**：帮助用户直观理解流量性价比
   - 基础 10G：¥3.0/GB
   - 畅享 50G：¥1.0/GB
   - 超值 100G：¥0.88/GB
   - 无限流量：不适用（不限量）
3. **标注推荐**：根据用户实际用量，标注哪个套餐更适合该用户
4. **说明差价**：明确告知月费差额，让用户清楚多花/少花多少钱

### 对比示例

```
畅享 50G vs 超值 100G：
- 月费：¥50 vs ¥88（差 ¥38/月）
- 流量：50GB vs 100GB（多 50GB，每GB仅 ¥0.76）
- 通话：500分钟 vs 1000分钟（多 500分钟）
- 权益：超值 100G 额外含【国内漫游免费】
→ 如果您月均用量超过 40GB 或经常出差，超值 100G 更划算
```

## 流量不足处理指引

> 用户反映流量不够用时的处理流程和加油包选项。

### 处理优先级

1. **先解决燃眉之急**：确认用户剩余流量，如接近零则优先推荐加油包
2. **再讨论长期方案**：分析用户近期用量趋势，建议套餐升级

### 加油包选项

| 加油包 | 价格 | 流量 | 有效期 | 适用场景 |
|-------|------|------|--------|---------|
| 日包 | ¥5 | 1GB | 当日有效（次日0点失效） | 临时急用，当天有大文件下载等 |
| 月包-小 | ¥15 | 5GB | 月底清零 | 本月偶尔超额，补充少量流量 |
| 月包-大 | ¥25 | 10GB | 月底清零 | 本月剩余天数较多，需较大补充 |

### 加油包购买方式

- APP → 流量加油包 → 选择对应包型 → 确认购买 → 立即生效

### 处理流程

```
1. 查询用户剩余流量（query_subscriber）
   ├─ 剩余流量 ≈ 0 且急需用网
   │   ├─ 推荐日包（1GB/¥5）应急
   │   └─ 距月底较远 → 推荐月包-小（5GB/¥15）或月包-大（10GB/¥25）
   └─ 尚有余量但经常月底不够
       └─ 跳过加油包，直接分析用量趋势

2. 分析近期用量趋势
   ├─ 本月属于偶发超额（如出差、假期）→ 建议加油包即可
   └─ 连续多月超额或用量持续增长 → 建议套餐升级
       ├─ 按月均流量匹配推荐套餐（参见"套餐推荐指引"）
       └─ 展示升级前后对比（参见"套餐对比指引"）

3. 用户确认
   ├─ 同意升级 → 引导办理（参见"套餐变更指引"）
   └─ 暂不升级 → 告知加油包随时可购，后续可再考虑升级
```

### 话术要点

- 先共情："理解您流量不够用的困扰"
- 先解决当下问题，再提长期方案
- 推荐加油包时说明有效期和价格，让用户自主选择
- 推荐套餐升级时用每GB单价对比，说明长期更划算
