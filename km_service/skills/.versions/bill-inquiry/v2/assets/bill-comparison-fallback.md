# 账单对比失败 fallback 模板

适用条件：

- `query_bill` 失败
- `analyze_bill_anomaly` 失败
- 缺少当前月或对比月账单
- 月份无法确定

推荐话术骨架：

```text
我已经识别到您是在问“账单对比/费用为什么上涨”，但这次没有成功拿到完整的对比数据，所以我现在不能准确告诉您是从多少变到多少，也不能直接判断是哪一项费用上涨。

我建议您让我重新查询一次，或者您稍后再试；如果问题比较着急，也可以联系 10086 进一步核查。
```

约束：

- 禁止出现任何未经成功查询验证的金额
- 禁止输出“增加了某项费用”“从 ¥X 到 ¥Y”这类具体结论
